도장 걸이 수량 시뮬레이션
주간 사용횟수 기반으로 박리량과 가용 걸이를 계산합니다
계산 방식 상세 설명
핵심 개념
  • 걸이 1개가 N회 사용되면 박리 필요 (수명 = 최대 사용 횟수)
  • 박리는 주 1회 발송, 1주 후 복귀
  • 많이 쓴 주 = 박리량 많음 / 적게 쓴 주 = 박리량 적음 (변동)
  • 공용 걸이: 색상 구분 없이 같은 걸이 사용
계산 로직
① 일 필요 걸이 = Σ 각 색상별 ⌈투입량 ÷ 장착수⌉
고정값 설정 색상은 고정값 사용 (예: WGR = 15개)
② 주간 총 사용횟수 = Σ(해당 주 각 일의 필요 걸이)
예) 70 + 35 + 58 + 88 + 49 = 300회
③ 주간 박리량 = ⌈주간 총 사용횟수 ÷ 수명⌉
예) ⌈300 ÷ 5⌉ = 60개 박리
④ 가용 걸이 = 총 걸이 − 주간 박리량
예) 110 − 60 = 50개 가용
⑤ 판정: 피크일 필요 걸이 ≤ 가용 걸이 → O 여유 / X 초과
⑥ 필요 총 걸이 = 피크일 필요 걸이 + 주간 박리량
예) 88(피크) + 60(박리) = 148개 필요
기존 방식과 차이점
기존: 박리량 = ⌈총 걸이 ÷ 수명⌉ → 고정값
수정: 박리량 = ⌈주간 사용횟수 ÷ 수명⌉ → 주마다 변동

물량이 많은 주는 박리량이 늘고 가용 걸이가 줄어드는 구조를 정확히 반영합니다.
